# Farkle — GroupGames101 rules summary

Reference summary from [Farkle Rules (and Scoring)](https://groupgames101.com/farkle-rules/) — GroupGames101. This write-up matches many “classic” tables but uses a **much higher straight** than most other online rule pages.

## Objective

Be the **first player to reach 10,000 or more** points (article wording). When someone reaches 10,000+, **remaining players each get one last throw** (per that article); if none match or beat that score, the player who hit 10,000 wins; if someone has more, **highest score wins**.

## Equipment

Six dice, cup, scoresheet, pencil; **2+** players (article suggests 3–6 is best).

## Setup

Choose first player however the group prefers (article suggests high roll on one die or “youngest first”).

## Turn flow

1. Roll all six dice.
2. Set aside **at least one** scoring die (or combination); you **need not** set aside every scoring die.
3. Re-roll **remaining** dice to build the turn total, or **stop** and add the turn total to your running game score.
4. If a roll scores **no** points, you **Farkle**: **lose all points accumulated that turn** (turn total goes to zero for that turn).

The article states combinations **must be from a single throw** (no “stacking” 3s across rolls to make three-of-a-kind). Multiple **different** combinations in **one** throw can all count (example given: 1 + three 2s + 5 = 350).

### Scoring combinations (as listed)

The article **only** lists scoring through three-of-a-kind plus straight and three pairs — **no** separate four/five/six-of-a-kind row on that page.

| Combination | Points |
|-------------|--------|
| Each **1** | 100 |
| Each **5** | 50 |
| Three **1**s | 1,000 |
| Three **2**s | 200 |
| Three **3**s | 300 |
| Three **4**s | 400 |
| Three **5**s | 500 |
| Three **6**s | 600 |
| Straight **1–2–3–4–5–6** | **3,000** |
| Three pairs | 1,500 |

## How this differs from other common write-ups

The **3,000**-point straight is **higher** than the **2,500** used in `FARKLE_RULES.md` (Cardgames.io) and the **1,500** straight in `FARKLE_RULES_PLAYMONSTER.md`. The endgame description uses **one last throw** per opponent rather than a full extra **turn** in some other rulesets — confirm at the table which you mean.
