# Farkle — “Standard scoring” (Wikipedia / Arnold)

Reference summary of the scoring table Wikipedia labels **standard scoring**, attributed to **Karen South Arnold**, *Playing Grandma’s Games* (Western Reflections, 2000). Wikipedia uses this table as a **baseline for comparison** across Farkle variants; it matches many folk and online implementations (e.g. same three-1s and straight values as [Cardgames.io](https://cardgames.io/blog/how-to-play-farkle/), documented in `FARKLE_RULES.md`).

**Sources:** [Wikipedia — Farkle](https://en.wikipedia.org/wiki/Farkle) (table and citation); [Internet Archive — *Playing Grandma’s Games*](https://archive.org/details/playinggrandmasg0000arno/mode/2up).

## Play (brief, as summarized on Wikipedia)

- Six dice; players take turns.
- Each turn: roll all dice, set aside **one or more** scoring dice, then either **bank** the turn total or **re-roll** the rest.
- **Hot dice**: all six dice score → may roll all six again (wording varies: some variants **must** re-roll).
- **Farkle** (no scoring dice on a roll): lose the **turn’s** accumulated points.
- When someone crosses the winning threshold (often **10,000**), **other players** usually get **one more turn** each; **high score** wins (common convention; always confirm).

Wikipedia also lists many **optional** regional rules (opening thresholds, piggybacking, five-dice variants, etc.) — see the article for those.

## Standard scoring table (Arnold / Wikipedia)

| Combination | Score |
|-------------|-------|
| Each **1** | 100 |
| Each **5** | 50 |
| Three **1**s | **1,000** |
| Three **2**s | 200 |
| Three **3**s | 300 |
| Three **4**s | 400 |
| Three **5**s | 500 |
| Three **6**s | 600 |
| Four of a kind | 1,000 |
| Five of a kind | 2,000 |
| Six of a kind | 3,000 |
| Three pair | 1,500 |
| Run (1–6) | **2,500** |

## Variations Wikipedia calls out

- **Commercial / Pocket Farkel–style** games often score **three 1s as 300** instead of 1,000 — see `FARKLE_RULES_PLAYMONSTER.md`.
- **Four+ of a kind** may be scored by **adding**, **doubling**, or **fixed** values depending on the group (Wikipedia gives examples).
