# Farkle – Zusammenfassung der GroupGames101-Regeln

Referenzzusammenfassung von [Farkle Rules (and Scoring)](https://groupgames101.com/farkle-rules/) – GroupGames101. Dieser Artikel entspricht vielen „klassischen“ Tabellen, verwendet jedoch eine **viel höhere Gerade** als die meisten anderen Online-Regelseiten.

## Ziel

Seien Sie der **erste Spieler, der 10.000 oder mehr Punkte erreicht** (Artikelwortlaut). Wenn jemand 10.000+ erreicht, erhalten **die verbleibenden Spieler jeweils einen letzten Wurf** (gemäß diesem Artikel); Wenn keiner diese Punktzahl erreicht oder übertrifft, gewinnt der Spieler, der 10.000 erreicht hat. Wenn jemand mehr hat, gewinnt **die höchste Punktzahl**.

## Ausrüstung

Sechs Würfel, Becher, Punkteblatt, Bleistift; **2+** Spieler (im Artikel wird empfohlen, dass 3–6 am besten sind).

## Aufstellen

Wählen Sie den ersten Spieler aus, wie es die Gruppe bevorzugt (im Artikel wird ein hoher Wurf mit einem Würfel oder „Der Jüngste zuerst“ empfohlen).

## Fluss drehen

1. Wirf alle sechs Würfel.
2. Legen Sie **mindestens einen** Wertungswürfel (oder eine Kombination davon) beiseite. Sie **müssen** nicht jeden Wertungswürfel beiseite legen.
3. Werfen Sie die **verbleibenden** Würfel erneut, um die Rundensumme zu ermitteln, oder **stoppen** Sie und addieren Sie die Rundensumme zu Ihrem laufenden Spielstand.
4. Wenn ein Wurf **keine** Punkte bringt, **Farkle**: **verlierst du alle in dieser Runde angesammelten Punkte** (die Rundensumme geht für diese Runde auf Null).

The article states combinations **must be from a single throw** (no “stacking” 3s across rolls to make three-of-a-kind). Multiple **different** combinations in **one** throw can all count (example given: 1 + three 2s + 5 = 350).

### Wertungskombinationen (wie aufgelistet)

Der Artikel listet **nur** die Wertung durch Drillinge plus Straight und drei Paare auf – **keine** separate Vierer/Fünf/Sechser-Reihe auf dieser Seite.

| Kombination | Punkte |
|-------------|--------|
| Jeweils **1** | 100 |
| Jeweils **5** | 50 |
| Drei **1**s | 1.000 |
| Drei **2**s | 200 |
| Drei **3**s | 300 |
| Drei **4**s | 400 |
| Drei **5**s | 500 |
| Drei **6**er | 600 |
| Gerade **1–2–3–4–5–6** | **3.000** |
| Drei Paare | 1.500 |

## Wie sich dies von anderen gängigen Zuschreibungen unterscheidet

Die **3.000**-Punkte-Straight ist **höher** als die **2.500**, die in „FARKLE_RULES.md“ (Cardgames.io) verwendet werden, und die **1.500**-Straight in „FARKLE_RULES_PLAYMONSTER.md“. Die Endspielbeschreibung verwendet **einen letzten Wurf** pro Gegner und nicht einen vollständigen zusätzlichen **Zug** in einigen anderen Regelsätzen – bestätigen Sie an der Tabelle, was Sie meinen.
