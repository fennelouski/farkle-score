# Farkle – PlayMonster / Pocket Farkel-Stil (Einzelhandelsübersicht)

Referenzzusammenfassung für **kommerzielle / Einzelhandels-** Farkle-Produkte (PlayMonster und ähnliche „Pocket Farkel“-Sets). Die Hausregeln variieren; Diese Datei entspricht der **drei 1er = 300**-Konvention, die in Wikipedia als typisch für kommerzielle Spiele bezeichnet wird, wobei andere Zeilen an der gemeinsamen „Standard“-Tabelle ausgerichtet sind, die in [FARKLE_RULES_WIKIPEDIA_ARNOLD.md](FARKLE_RULES_WIKIPEDIA_ARNOLD.md) für Vielfache, Gerade und drei Paare verwendet wird, sofern in Ihrer Box nichts anderes angegeben ist.

## Bewertung (App-Engine)

| Melden | Punkte |
|------|--------|
| Jeweils **1** | 100 |
| Jeweils **5** | 50 |
| Drei **1**s | **300** |
| Drei **2**s | 200 |
| Drei **3**s | 300 |
| Drei **4**s | 400 |
| Three **5**s | 500 |
| Drei **6**er | 600 |
| Vierling | 1.000 |
| Fünfling | 2.000 |
| Sechsling | 3.000 |
| Drei Paare | 1.500 |
| Gerade **1–2–3–4–5–6** | 2.500 |

## Quellenhinweise

- Wikipedia – *Farkle*: Kommerzielle Spiele erzielen oft **drei Einsen als 300** statt 1.000.
- Für vollständige Turn-Regeln verwenden Sie Ihre Box-Einlage oder einen anderen mitgelieferten Regelsatz.
