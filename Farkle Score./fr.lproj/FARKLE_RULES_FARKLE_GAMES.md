# Farkle — Résumé des « règles officielles » de Farkle.games

Résumé de référence des [Règles officielles Farkle](https://farkle.games/official-rules/) — **farkle.games** (site qui publie les applications mobiles Farkle). Le score pour **trois paires** et la **straight** est **inférieur** à celui de nombreuses tables folk/vente au détail ; la page documente également une pénalité **triple-Farkle**.

## Objectif

Avoir le **score le plus élevé supérieur à 10 000** lors du **dernier tour** de jeu. Lorsqu'un joueur atteint **10 000+**, ce joueur « sort » ; **chaque** autre joueur a **un tour de plus** pour battre le meilleur score (la page le note **varie selon les familles et les amis**).

## Tourner le flux

- Lancez six dés, mettez de côté les combinaisons de notation.
- Après chaque lancer, **au moins un** dé de score doit être sélectionné.
- Soit **comptez** les points du tour dans votre total et passez, soit **relancez** les dés restants.
- **Farkle** : aucun dé de score sur un lancer → **perdez tous les points gagnés ce tour** (tour).
- **Dés chauds** : les six dés utilisés pour marquer → lancer **tous les six à nouveau** ; peut se répéter. (Sous ** Variations de gameplay **, le site indique que certains groupes nécessitent une relance obligatoire lorsque les six marquent.)

## Score par rouleau (graphique principal)

Les dés de **différents lancers** ne peuvent pas être combinés en une seule combinaison (le site illustre 5-5 puis 5 comme trois simples, pas trois 5).

| Combinaison | Points |
|-------------|--------|
| Chaque **1** | 100 |
| Chaque **5** | 50 |
| Brelan | **Valeur nominale × 100** (par exemple 2-2-2 = 200) |
| Trois **1** | **1 000** (exception au modèle ×100) |
| Trois paires | **750** |
| Droit **1–2–3–4–5–6** | **1 000** |

La page principale répertorie les scores pour les simples, les brelans, les trois paires et la quinte ; il ne met **pas** quatre/cinq/six d'un genre dans le tableau principal.

## Triple pénalité de Farkle

Si vous **Farkle trois fois de suite**, vous **perdez 1 000 points** de votre total en banque (« Aïe ! »). La section **Variations de gameplay** indique que certains groupes utilisent une pénalité de **500** ou **aucune**.

## Variations facultatives (répertoriées sur la même page)

Le site documente les règles internes courantes, notamment : les scores **d'effraction** (par exemple 300 pour « entrer sur le tableau ») ; trois paires ont obtenu une note de **0/500/750/1 500** ; **trois as** comme 1 000 contre 100 ; **4/5/6 d'une sorte** en tant que multiples du score du brelan ; droit **1 000 à 1 500** ; jouez jusqu'à **5 000** ou **20 000** ; **deux triplés** pour **2 500** ; **superposition** en utilisant les dés restants du joueur précédent.
