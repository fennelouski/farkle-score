# Farkle — Style PlayMonster / Pocket Farkel (résumé de la vente au détail)

Résumé de référence pour les produits Farkle **commerciaux/au détail** (PlayMonster et ensembles similaires « Pocket Farkel »). Les règles de la maison varient ; ce fichier correspond à la convention **trois 1 = 300** décrite dans Wikipédia comme étant typique pour les jeux commerciaux, avec d'autres lignes alignées sur le tableau « standard » commun utilisé dans [FARKLE_RULES_WIKIPEDIA_ARNOLD.md](FARKLE_RULES_WIKIPEDIA_ARNOLD.md) pour les multiples, les lignes droites et trois paires, sauf indication contraire dans votre boîte.
