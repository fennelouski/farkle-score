# Zilch — Résumé des règles de Playr.co.uk

**Zilch** est de la même grande famille que Farkle (poussez votre chance avec six dés, « perdez le tour » sur un lancer mort). Les règles diffèrent dans les détails. Ce fichier résume les [Rules of Zilch](https://zilch.playr.co.uk/rules.php) sur **playr.co.uk** (2007-2008).

Utilisez-le lorsque vous souhaitez **doubler les calculs du carré**, un seuil bancaire de **300** ou la pénalité de **troisième zilch** - ce qui n'est pas identique aux règles encadrées typiques de Farkle.

## Tourner le flux

1. Lancez les six dés pour commencer le tour.
2. Si un dé marque un score, vous **devez prendre certains** de ces points avant de relancer.
3. Si **aucun** résultat de dé → vous **zilch** : **effacez** les points pris jusqu'à présent ce tour ; le tour se termine. **Troisième zilch consécutif** → **perdez 500** de votre score en banque.
4. Une fois que vous avez pris **au moins 300** points ce tour-ci, vous pouvez **mettre en banque** et terminer le tour, ou continuer à lancer les dés.
5. Relancez tous les dés que vous **n'avez pas** utilisés dans la ou les combinaisons de notation choisies.
6. Si vous marquez avec **tous les six** dés → **lancement gratuit** des six (même idée que les « dés chauds »).

## Guide de notation (de Playr)

- **1s** : 100 chacun ; **5s** : 50 chacun.
- **Brelan** : 1s = **1 000** ; 2s-6s = **200-600** (motif visage × 100, identique à celui du Farkle commun).
- **Quatre ou plus** : commencez à partir du score du brelan, puis **doublez pour chaque dé supplémentaire** au-delà de trois (exemples de Playr : quatre 2 → 400 ; cinq 2 → 800 ; six 3 représentés par 300 → 600 → 1 200 → 2 400 dans leur chaîne d'illustrations).
- **Lancers spéciaux** (chacun accorde également un **lancement gratuit** par joueur) :
- Droit **1–6** (toute commande) : **1 500**
- **Trois paires** : **1 500**
- **Aucun dé de notation** sur un lancer : **500** et un **lancement gratuit** (leur exemple « spécial » — inhabituel contre un zilch normal ; lisez leur guide de notation sur le site pour le traitement exact).

**Une combinaison par dé :** Les dés ne peuvent compter que pour **une** combinaison de notation sur ce lancer.
