# Farkleの遊び方

[Cardgames.io — How to play Farkle](https://cardgames.io/blog/how-to-play-farkle/) を元にした参考要約です。Farkleには地域ごとのバリエーションがありますが、ここではこのサイトで使われているシンプルなルールセットを説明します。

## 目的

**10,000 ポイント以上を貯めた**最初のプレイヤーになりましょう。ターンは交互に行われ、プレイヤーはメルドを獲得し、ローリング (バンク) をいつ停止するかを決定するか、「ファークル」のリスクを冒してラウンドのバンクされていないポイントを失うかを決定します。

## 用具

標準的なサイコロが 6 つあります。

## ターンの流れ: ロール、スコア、バンク

1. ターンの開始時に、**6 つのサイコロをすべて振る**。
2. 各ロールの後、得点メルドを形成するダイスを **脇に置きます** (以下を参照)。各ロールごとに**少なくとも 1 つ**の得点ダイスを確保しなければなりません。さらに、どの得点ダイスを保持し、どれを再ロールするかを選択します。
3. 取っておいていないダイスを**再ロール**して、さらに多くのメルドを獲得するか、**バンク**してラウンドの合計を永久スコアに追加し、ターンを終了します。バンクポイントは失われることはありません。
4. サイコロを相手に渡します。彼らの番が始まります。

### 役はその1投ごとに判定

メルドは**ロール間で引き継がれません**。別々のスローで得点されたダイスは別々のメルドです。

例: 1を2個出して脇に置くと (100 + 100 = **200**)。次のロールでさらに1が出ても、それは**同じ投擲で出た3個の1**ではないため1,000点にはなりません。追加の1は単独で**100**点となり、合計は**300**点です。

## ファークル

ロール後、**どのダイス**も得点役に使えない場合は**ファークル**です。ターンは即終了し、そのラウンドで**まだバンクしていない点数はすべて失います**。その後、手番は相手に移ります。振り直しを重ねるほどファークルのリスクは上がります。

## ホットダイス

振ったばかりの **すべてのダイス** がメルドに使用できる場合 (6 つすべてのダイス スコア)、**ホット ダイス** を持っています。 **6 つ**すべてのサイコロを手に取り、もう一度振り、同じターンを続けることができます。このラウンドですでに獲得したポイントは、バンクまたはファークルするまで「リスクのある」プールに残ります。

## ゲーム終了

プレイヤーが **10,000 ポイント以上のポイントを貯めて **ターンを終了**すると、ゲームは終了します。勝てたらすぐに銀行に預ける必要はありません。テーブルが許せばローリングを続けることもできますが、勝つためにバンキングすることが通常の目標です。

## 得点メルド

|メルド |ポイント |
|------|----------|
|各 **1** | 100 |
|各 **5** | 50 |
| 3 つの **1** | 1,000 |
| 3 つの **2** | 200 |
| 3 つの **3** | 300 |
| 3 つの **4** | 400 |
| **5** が 3 つ | 500 |
| **6** が 3 つ | 600 |
| **4** の種類 | 1,000 |
| **5個同じ目** | 2,000 |
| **6** の種類 | 3,000 |
| **3 ペア** (フォーカード + ペアを含む) | 1,500 |
| **ストレート** 1–2–3–4–5–6 | 2,500 |
