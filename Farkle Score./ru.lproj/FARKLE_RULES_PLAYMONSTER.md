# Farkle — PlayMonster / Pocket Farkel–style (retail summary)

Reference summary for **commercial / retail** Farkle products (PlayMonster and similar “Pocket Farkel” sets). House rules vary; this file matches the **three 1s = 300** convention called out in Wikipedia as typical for commercial games, with other rows aligned to the common “standard” table used in [FARKLE_RULES_WIKIPEDIA_ARNOLD.md](FARKLE_RULES_WIKIPEDIA_ARNOLD.md) for multiples, straight, and three pairs unless your box says otherwise.
